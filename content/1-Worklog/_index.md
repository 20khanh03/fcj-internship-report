This page summarizes the entire development process of an AI-powered English learning application over a period of **8 weeks** (from 09/03/2026 to 08/05/2026).

The project was built using a **serverless architecture on AWS**, focusing on leveraging managed services to reduce development time and optimize costs.

During the development process, the project was divided into the following main stages:

**Week 1:**  
[Overview of AWS, account setup, introduction to IAM, AWS Console, and core services.](1.1-week1/)

**Week 2:**  
[Hands-on practice with EC2, S3, and VPC.](1.2-week2/)

**Week 3:**  
[Experimenting with deploying a web application on EC2, exploring databases, and starting frontend/backend setup.](1.3-week3/)

**Week 4:**  
[Building the system foundation.](1.4-week4/)

**Week 5:**  
[Developing the AI Tutor and Flashcard SRS features.](1.5-week5/)

**Week 6:**  
[Improving AI Tutor and Flashcard, and further system integration.](1.6-week6/)

**Week 7:**  
[Integrating real-time voice features and building the learning progress dashboard.](1.7-week7/)

**Week 8:**  
[End-to-end testing, bug fixing, and system deployment.](1.8-week8/)
